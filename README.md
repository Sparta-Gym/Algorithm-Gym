+ 챕터마다 알게된 점 README에 적어놓기
+ 풀이가 생각안나서 답안 본 문제는 표시해놓고 일정 기간이 지난 후 다시 풀어보기
  + 문제번호[x].cpp로 체크
+ 1주에 1강 내지는 2강 진도 나가기
  + 연습 문제가 많은 경우는 1주 1강
  + 연습 문제 끝나는 대로 다음 강의 듣기
+ 부족하다고 느끼는 단원은 연습문제 이외에 따로 더 풀기
+ 복잡한 문제의 경우 문제번호.md 파일 만들어서 풀이과정 정리해놓기
